## Como usar este arquivo

Quando o Editor produz um novo episódio, ele consulta este arquivo para aplicar lições de episódios anteriores. Cada lição tem:
- **O que aconteceu:** contexto do episódio
- **O que funcionou / não funcionou:** observação concreta
- **A regra derivada:** o que mudar na produção futura
